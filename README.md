== Aplicação para controle de almoxarifado.


# Instalação


```ruby
bundle install
rake db:create
rake db:migrate
rake db:seed, para criar usuario com login 'bernardo' e senha 'bernardo123'.
rails s
acessar browser no endereço 'localhost:3000'
```

## Detalhes técnicos:

Versão Ruby: 2.2.3
Versão Rails: 4.2.6
